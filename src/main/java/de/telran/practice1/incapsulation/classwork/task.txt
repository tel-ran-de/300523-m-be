Задача: Создать класс Animal с полями name, age в соответствии с принципом инкапсуляции. Понять как модификаторы доступа
влияют на доступ к полям и методам класса.

Необходимые действия:
1.Создать класс Animal c двумя публичными полями name, age.
2.Создать конструктор для класса Animal c двумя полями
3.Создать публичные getters, setters для полей класса
4.Создать класс Main с методом public static void main()
5.Создать объект класса Animal в классе Example
6.Вывести в консоль значения полей name, age используя getters (например animal.getName())
7.Вывести в консоль значения полей name, age используя обращение к полю класса (например animal.name)

Что бы понять как работают модификаторы доступа необходимо изменить модификаторы доступа у полей, методов, а так же попробовать перенести
класс Animal в другой package относительно того, где расположен метод public static void main()

Что менять:
1. Изменить модификатор доступа у полей класса c public на default, на protected, на private.
Запустить код, посмотреть, что происходит. Объяснить почему такое поведение.
2. Изменить модификатор доступа у getters, setters c public на default, на protected, на private.
Запустить код, посмотреть, что происходит. Объяснить почему такое поведение.
3. Создать новый package, переместить в него класс Animal. Запустить программу. Посмотреть что происходит.
Поменять модификаторы доступа у полей и методом класса Animal, посмотреть, что происходит и почему.
