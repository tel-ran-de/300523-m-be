В этой задаче мы напишем механизм, который позволит нам сортировать объекты по нашим условиям.

1. Создать класс Person с полями: name, age
2. Создать класс Main внутри которого создать несколько экземлпяров Person
3. Добавить созданные объекты в массив
4. Создать компаратор для сравнения объектов, использующий два поля класса Person таким образом,
что если имена разные - отсортировать по имени, если имена одинаковые, то отсортировать по возрасту
4.1* Те, кто возможно уже знают иной способ реализации компоратора через methodReference - могут написать с ним
5. Отсортировать массив класса Person
